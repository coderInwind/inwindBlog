  ## 闭包的定义
闭包指的是一个函数对其周围状态的引用捆绑在一起，这样的组合称为闭包。
通俗的来讲:
```
function foo(){
	var name = "你的名字是？"
	return function bar（）{
		console.log(name);
	}
}

let fn = foo()

fn()
```
一个函数，访问着外层的自由变量变量，这样就形成了一个闭包
根据js内存回收机制（如上一个博客），没有被根指向的都会被销毁

## 内存泄漏
foo的AO对象被bar的对象的parentScope指向，所以bar函数对象永远都不会被销毁，这种情况被称为内存泄漏
如何消除内存泄漏呢？只需要fn = null 把fn指向另一个地方即可
